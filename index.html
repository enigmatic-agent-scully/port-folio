about aj

AJ is a 29-year-old Full-Stack Web Developer based in Atlanta, Georgia. Her dream job would challenge her wide variety of skills, including web development, office administration, research and analysis, and creative and advertorial writing and design, in addition to providing a cornucopia of opportunities for growth and learning.

AJ particularly enjoys learning about technology, world history, human rights, sociology, psychology, intersectional feminism, popular culture analysis, art techniques, and the interconnectedness of the universe.

AJ's hobbies include reading, hand lettering, wood carving, fiber crafts, collage, printmaking, tv watching, and code-writing.

projects

volume_up Multi-User Noise Manipulator

This site allows the user to produce and manipulate the frequency and gain (volume) of an audio signal, as well as listen to other users' audio in real time. This project employed the WebAudio API to create the oscillators, Google Firebase to store and retrieve other users' mouse coordinates for use in producing audio, and Materialize for the layout.

directions_transitTrain Time

This is a simple train schedule application that incorporates HTML, Bootstrap, JavaScript, jQuery, Google Firebase, and Moment.js. Based on the user's input, it calculates when the train will next arrive, as well as how many minutes until it arrives.

local_atm Rupee Collector

Inspired by Zelda, this fun game uses challenges players to figure out the random values assigned to four different color rupees and use that knowledge to add the values until they equal a randomly chosen number.

Bamazon

This app uses Node.js and MySQL to create a command line interface (CLI) that allows the customer to either browse all items in the MySQL database or browse by department and place an order based on the item id. The quantity of each item ordered is then automatically deducted from the item's total quantity in the database. There is also a Manager's Portal which allows for additional functionality.

Friend Finder

This app uses Node.js, Express, Materialize, and other npm packages to take in form data, compare the data to other users' data, and return the closest matching user.
